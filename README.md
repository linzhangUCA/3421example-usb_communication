# USB Communication Example

## Usage
Connect Pico with your computer via a **data capable** Micro-USB cable.
1. Save [pico_messenger.py](pico_messenger.py) on a Pico board as `main.py`.
2. Hard reset(replug) Pico.
3. Run [computer_messenger.py](computer_messenger.py) on your PC/desktop/laptop/SBC.

> [!NOTE]
> Change the `SERIAL_PORT` in [computer_messenger.py](computer_messenger.py) according to your operating system type.
